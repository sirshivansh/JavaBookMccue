## Learning Points.

1. to print we use:

void main(){
    IO.println("Hello");
}

2. to run we use:

java Main.java or java src/Main.java (if not in current working directory)

3. In modern Java, we need to use " java -cp location filename"
 ex: java -cp code\src comments
to be able to run those class files. Here -cp means "Look for the class files"
